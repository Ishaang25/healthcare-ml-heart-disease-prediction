# Heart Disease Prediction using Machine Learning

## Objective
Build a machine learning model to predict the presence of heart disease from patient clinical data.

## Dataset
- UCI Heart Disease Dataset
- 302 unique observations after removing duplicate records

## Workflow
1. Data loading
2. Exploratory Data Analysis (EDA)
3. Duplicate detection and removal
4. Train-test split
5. Logistic Regression
6. Model evaluation

## Results
- No missing values
- 723 duplicate rows identified and removed
- Logistic Regression Accuracy: ~77%
